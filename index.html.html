Featured Projects

AI + Computer Vision + Robotics

AI-Driven Vision Guided Laser based QR Encoding Using Adaptive Clean-Zone Localization for Railway Track Components

Intelligent contactless railway inspection system that detects rusted and clean metallic surfaces on fishplates using transformer-based semantic segmentation. The system calculates rust severity percentage, identifies the largest clean-metal region and converts pixel coordinates into angular offsets to guide an ESP32-powered pan-tilt laser turret for precise QR encoding. Enables automated infrastructure monitoring, digital condition tracking and predictive maintenance while reducing manual inspection errors.

Research Work

Power Systems + Deep Learning

Future Power System Inertia Prediction Using Deep Learning–Based Time-Series Models

Developed a time-series forecasting framework to predict future power grid inertia under renewable energy integration. Implemented LSTM, GRU and BiLSTM models and performed multi-step forecasting to analyze long-term frequency stability behaviour. Results demonstrate improved prediction stability using LSTM for proactive grid reliability assessment in low-inertia power systems.

Physics-Guided Modeling

System Inertia Estimation Using Frequency Measurements

Proposed a measurement-based inertia estimation method derived from the swing equation using ROCOF extracted via Savitzky–Golay filtering. The framework provides physically interpretable real-time inertia monitoring without training data and demonstrates robust estimation across multiple operating conditions.
